 Grade Calculator using Streamlit

The Grade Calculator is a simple and interactive web application developed using Python and Streamlit. This project is designed to help students and educational institutes calculate academic percentages quickly and accurately. Users can enter the institute name, student name, and obtained marks to instantly generate the percentage and corresponding grade.

The application provides a clean and user-friendly interface, making it easy to use for students, teachers, and academic staff. By automating percentage calculations, the project reduces manual effort and minimizes calculation errors. The system demonstrates the practical use of Python programming and Streamlit for building responsive educational applications.

This project is suitable for academic learning, portfolio building, and final-year project demonstrations. It showcases the implementation of form handling, data processing, and real-time result generation in a modern web application environment.

Key Features

- Institute Name Input
- Student Name Input
- Marks Entry System
- Automatic Percentage Calculation
- Instant Grade Generation
- Interactive Streamlit Interface
- Fast and Accurate Results

Technologies Used

- Python
- Streamlit

This project aims to simplify academic performance evaluation while providing an efficient and user-friendly experience.
